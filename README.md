# Macrotb

## Macro para aquisição de itens, completamente implementada em linguagem de programação C++

https://github.com/viniimiguel/Macrotb/assets/144070822/79effa76-5cc5-4863-84a1-e94167c9a4cb


## Modo de Utilização: Para empregar esta ferramenta, o usuário necessita primeiramente estabelecer o valor de (SQM) referente à posição dos pés do seu personagem.
## Lembrem-se de deixar definido no jogo para pegar os loots com a tecla SHIFT + RIGHT.

### Para fazer isto basta apenas apertar a tecla "F" no (SQM) referente ao pé isso ja vai ser definido automaticamente.

![image](https://github.com/viniimiguel/Macrotb/assets/144070822/c16ce9bf-4659-4cf7-ace2-9fe79ef03fd9)

### Após a definição desse parâmetro, é necessário atribuir uma tecla específica para ativar o macro.

![image](https://github.com/viniimiguel/Macrotb/assets/144070822/a95ad856-d538-4903-8b86-e407ff93d96e)

### A tecla deve ser definida com valores em hexadecimal como por exemplo "0x46" que e o valor referente a tecla "F", Abaixo irei deixar os valores referentes a cada tecla.
### Todas as teclas estarão dentro do arquivo .rar no projeto.
